# Embedded Security System

An Arduino-based Embedded Security System that combines RFID authentication and keypad password verification to provide enhanced access control.

## Features

- RFID Card Authentication
- Password Verification using Keypad
- LCD Display Interface
- Servo Motor Controlled Lock
- Buzzer Alarm for Unauthorized Access
- LED Status Indicators

## Components Used

- Arduino UNO
- RFID Reader (MFRC522)
- RFID Tags/Cards
- 16x2 LCD with I2C Module
- 4x4 Matrix Keypad
- Servo Motor (SG90)
- Buzzer
- LEDs
- Breadboard
- Connecting Wires

## Working Principle

1. User scans RFID card.
2. System verifies RFID UID.
3. If valid, user enters password.
4. Password is verified.
5. Servo motor unlocks the door.
6. Wrong RFID or password triggers buzzer and red LED.
